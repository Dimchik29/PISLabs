## Обзор проекта

Данный репозиторий содержит материалы учебного проекта по дисциплине «Прикладные интеллектуальные системы».

В рамках проекта было разработано программное приложение **APACHE4 — симулятор автономного гидрографического катера**. Программа предназначена для демонстрации принципов автономного движения роботизированного катера по акватории, выполнения маршрута сканирования, сбора телеметрии, имитации работы датчиков и экспорта результатов обследования.

Приложение реализовано как настольный симулятор на языке **Python** с использованием библиотеки **Pygame**. Оно позволяет наглядно показать работу автономного катера без необходимости использования реального оборудования.

---

## Цель проекта

Цель проекта — разработать учебное программное приложение, моделирующее работу автономного гидрографического катера APACHE4 при выполнении обследования водоёма.

Программа должна демонстрировать:

* ручное управление катером;
* автоматическое движение по маршруту;
* построение маршрута типа «змейка»;
* возврат катера на базу;
* удержание текущей позиции;
* имитацию работы эхолота;
* имитацию ADCP-датчика;
* имитацию RTK GNSS;
* отображение телеметрии;
* запись и экспорт данных измерений.

## Описание интерфейса

Интерфейс приложения состоит из нескольких основных областей:

1. **Основное поле симуляции**
   Отображает водоём, катер, маршрут, базовую точку, траекторию движения и точки измерений.

2. **Панель телеметрии и управления**
   Содержит данные о скорости, курсе, глубине, GPS, связи, заряде батареи, двигателях и состоянии датчиков.

3. **Карта маршрута**
   Показывает уменьшенную схему движения катера по акватории.

4. **Профиль глубин**
   Отображает график изменения глубины по последним измерениям.

5. **Журнал событий**
   Фиксирует основные действия: запуск автопилота, включение записи, экспорт данных, возврат домой и другие события.

## Экспорт результатов

Приложение позволяет сохранять результаты работы в папку `outputs`.

При нажатии кнопки **«Экспорт»** создаются два файла:

* CSV-файл с измерениями;
* JSON-файл с кратким отчётом о миссии.

CSV-файл содержит:

* время измерения;
* координаты катера;
* скорость;
* курс;
* глубину;
* температуру;
* заряд батареи;
* статус GPS;
* параметры течения.

JSON-файл содержит общую информацию о выполненной миссии:

* режим работы;
* количество точек маршрута;
* количество записанных измерений;
* площадь покрытия;
* заряд батареи;
* последнюю измеренную глубину;
* статус GPS;
* список реализованных функций.
