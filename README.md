## Overview

This repository contains the laboratory work developed as part of the course "Applied Intelligent Systems." The project focuses on creating software for autonomous water body scanning using the Apache4 robotic boat.

**Objective:** Study the hardware structure, main components, and interfaces of the Apache4 robotic boat...

**Key Components Studied:**
- Hull and Propulsion System

- Communication interfaces

### [Laboratory Work No. 3 – Navigation Design](https://github.com/Jewgien/Dev-of-the-robot-Keenon/blob/main/Laboratory%20work%20%E2%84%963%20PIS.docx)

**Topic:** Design of Autonomous Navigation Parameters and Water Body Scanning Route Maps

**Objective:** Learn to configure the Apache4 robotic boat, modify motion parameters, and create route maps...

**Key Tasks:**
- Interface and telemetry settings
- Sound and notification configuration
- Creating route maps with Mapping Tool
- Waypoint placement (Base, Scan, Return, Obstacle)
