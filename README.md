# Development of Software for Autonomous Waterbody Scanning Using Apache4 Robotic Boat

## Project Overview

This repository contains laboratory works completed as part of the course **Applied Intelligent Systems**.

The project is dedicated to the development of software for autonomous scanning and monitoring of water bodies using the Apache4 robotic boat.

## Laboratory Works

### Laboratory Work №1

Project Planning for Autonomous Waterbody Scanning using Apache4.

Topics:

* Project architecture
* Development stages
* Navigation planning
* Scanning workflow design

### Laboratory Work №2

Analysis of Apache4 Hardware Platform.

Topics:

* Hardware architecture
* Navigation modules
* Sonar systems
* Telemetry and communication
* Sensors and interfaces

### Laboratory Work №3

Design of Autonomous Navigation Parameters and Mapping System.

Topics:

* Route planning
* Waterbody mapping
* Scan point configuration
* Safety zones
* Navigation algorithms

### Laboratory Work №4

Implementation of Autonomous Operating Modes.

Topics:

* Autonomous Patrol Mode
* Waypoint Navigation Mode
* Obstacle Avoidance Mode
* Return To Base Mode
* Monitoring & Broadcast Mode

## Technologies

* Autonomous Navigation
* GPS / GNSS
* Sonar Mapping
* Telemetry Systems
* Intelligent Control Systems
* Waterbody Monitoring
