# APACHE4 Exam Ready Simulator

Готовое десктоп-приложение на Python для демонстрации автономного сканирования водоёмов роботизированным катером Apache.

## Что исправлено

В этой версии полностью переписан автопилот:

- исправлена ошибка направления поворота;
- курс считается в одной системе координат: `0° = вправо`, `90° = вниз`;
- автопилот плавно поворачивает к целевой точке;
- катер не крутится вокруг waypoint;
- маршрут проходится по точкам последовательно;
- возврат домой останавливает катер на базе и переводит его в режим HOLD.

## Возможности

- ручное управление катером;
- автопилот по маршруту «змейкой»;
- возврат домой;
- удержание позиции;
- имитация эхолота;
- имитация ADCP;
- имитация RTK GNSS;
- телеметрия: скорость, курс, глубина, батарея, связь, крен/дифферент;
- карта маршрута и фактического трека;
- запись точек промера глубины;
- график профиля глубин;
- экспорт CSV и JSON в папку `outputs`.

## Запуск

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

Или на Windows можно запустить:

```bash
run.bat
```

## Управление

- `W` / стрелка вверх — увеличить скорость
- `S` / стрелка вниз — назад / тормоз
- `A` / стрелка влево — поворот влево
- `D` / стрелка вправо — поворот вправо
- `Space` — стоп
- `P` — автопилот
- `H` — удержание позиции
- `B` — возврат домой
- `R` — запись измерений
- `E` — эхолот
- `C` — ADCP
- `G` — RTK GNSS
- `M` — новый маршрут
- `X` — экспорт данных
- `Backspace` — сброс симуляции
- `Esc` — выход

## Как показывать на сдаче

1. Запустить `python main.py`.
2. Нажать `Маршрут`.
3. Нажать `Автопилот`.
4. Показать, как катер проходит маршрут и записывает глубины.
5. Нажать `Домой`, чтобы показать возврат на базу.
6. Нажать `Экспорт`, чтобы показать сохранение CSV/JSON.
